函数 ​

函数是将具有某一块特定功能的代码块封装起来，方便重复使用。

为什么使用函数？

  • 代码简洁
  • 一次封装，多次使用
  • 根据不同参数可以返回不同结果

声明方式 ​

  • 具名函数
js
function 函数名(参数){
    //函数体
}
  • 匿名函数
js
var 变量名 = function(参数){
    //函数体
}
  • 实例化构造函数
js
var 变量名 = new Function( "参数1" , "参数2" ... , "函数体");//括号里的参数是字符串形式的
  • 箭头函数
js
// 相当于：(参数1, 参数2, …, 参数N) =>{ return 表达式; }
(参数1, 参数2, …, 参数N) => { 函数声明 }
(参数1, 参数2, …, 参数N) => 表达式(单一)

//当只有一个参数时，圆括号是可选的：
(单一参数) => {函数声明}
单一参数 => {函数声明}

//没有参数的函数应该写成一对圆括号:
() => {函数声明}

//实例
   // ES5
var x = function(x, y) {
     return x * y;
}
   // ES6
const x = (x, y) => x * y;

//注意：箭头函数没有局部this的绑定
//和一般的函数不同，箭头函数不会绑定this。 或者说箭头函数不会改变this本来的绑定。

调用方式 ​

  • 通过括号方式调用
js
//具名函数调用
函数名();
//匿名函数调用
变量名();
  • 自调用
js
//第一种
(function(){
  //函数体
 })();

//第二种
(function(){
  //函数体
 }());

//第三种
!function(){
    //函数体
}();

注意：自调用函数前面必须加分号

  • 通过事件的方式调用
html
<div class="box" onclick="fn()"></div>;
<script>
    function fn(){
        alert(1);
    };
</script>

注意事项 ​

  • 函数名不可以重复，后面的会把前面的覆盖掉
js
function fn(){
    console.log(1)
};
function fn(){
    console.log(2)
};
fn();  //输出为2
  • 具名函数可以在函数前后调用，匿名函数只能在函数后面调用
js
//具名函数
fn();
function fn(){};
fn();

//匿名函数
fn();       //失败
var fun = function(){};
fn();
  • 在不同的script代码块中，不管是具名函数还是匿名函数只能在函数后面的代码块调用
html
<script>
    fn();   //报错 fn is not defined
    fun();  //报错 fun is not defined
</script>
<script>
    function fn(){};
    var fun = function(){};
</script>
<script>
    fn();   
    fun();
</script>

参数 ​

函数的参数：

  • 实参：函数调用的时候传入的参数
  • 形参：函数声明的时候接收的参数
js
function fun (num){    // num 形参
    console.log(num);
}
fun(100)  //100  实参

函数参数注意事项：

  • 参数可以是任意数据类型

  • 参数的个数，多个参数用逗号隔开

    • 实参和形参是一一对应的

    • 当形参大于实参，那么多余的形参显示undefined

      js
      function fn(num,name){
    console.log(num);  // 100
    console.log(name); // undefined
}
fn(100);

    • 当形参小于实参

      函数声明的同时会自动创建一个arguments对象，用来接收函数参数。(只能在函数中使用)

      js
      function fn(num){
    console.log(num);   //100
    console.log(arguments);
    console.log(arguments[0]); //100
    console.log(arguments[1]); //张三
    console.log(arguments[2]); //true
}
fn(100,"张三",true);

函数返回值 ​

函数返回值return

返回值作用：

  • 终止当前函数
  • 返回函数的结果

返回值说明：

  • 每一个函数都有返回值，如果没有返回值默认返回undefined
  • 函数返回值可以是任意数据类型
  • 一个函数只能有一个返回值，如果有多个只返回最后一个
  • 声明一个变量赋值为该函数调用，那么这个函数就是该函数的返回值
  • 同上 console.log(函数调用)，那么输出该函数的返回值
js
function fn(){}
console.log(fn());   /// undefined

function fun(){
    return 100;
}
console.log(fun()); // 100

作用域 ​

作用域即一段代码的作用范围。

环境：

  • 宿主环境：浏览器

  • 执行环境

    • 全局环境

      全局环境是在全局环境下使用var声明的变量。

      全局环境声明的变量后面不管是全局环境还是局部环境都可以访问的到。

    • 局部环境（函数环境）

      局部变量是在函数环境声明的变量。参数也是局部变量。

      局部环境声明的变量或者函数只能在当前环境或者子环境中使用。

作用域链 ​

当访问一个变量或者函数的时候，会在当前环境中找，当前环境中找不到回去上一个环境中去找，上一个环境找不到再去上一级环境中找，直到找到全局环境（window）为止，变量或者函数找不到会报错：变量或者函数is not defined。

js
var num = 1;
function aa(){
  var num = 2;
  function bb(){
    var num = 3;
    function cc(){
      num = 4;
      alert(num);    // 4
    }
    cc();
    alert(num);        // 4
  }
  bb();
  alert(num);            // 2
}
aa();
alert(num);                // 1
//注意：
//不用var声明直接为变量赋值，得到的不是真正的全局变量，它实际是对属性赋值操作。首先，它会尝试在当前作用域链中解析 改变量； 如果在任何当前作用域链中找到改变量，则会执行对改变量属性赋值； 如果没有找到改变量，它才会在全局对象（即当前作用域链的最顶层对象，如window对象）中创造改变量属性并赋值。

预解析 ​

  • 代码从上到下执行，不同的 script代码块依次执行

  • 遇到var或者function会提前解析到内存中

    js
    console.log(fun);   // 函数fun
var fun = 123;
function fun() {
    console.log('Hello javascript');
}
console.log(fun);     // 123

//以上这段代码中的真正执行顺序为
//var fun => function fun(){} => console.log(fun) => fun=123 => console.log(fun)

  • 首先运行全局环境，函数环境调用的时候再执行

回调函数 ​

回调函数是将一个函数当作另一个函数的参数，那么这个函数就是回调函数。

js
function aa(){
    fn();     //调用回调函数bb
    console.log(fn());   //获取回调函数的返回值 undefined
}
function bb(){
    console.log("回调函数")
}
aa(bb);       //bb回调函数

给回调函数传递参数

js
function fn(){
    //arguments[0] 回调函数
    arguments[0]("张三");
}
fn(function(name){
   console.log(name);   //张三
   });

递归函数 ​

在函数中调用它自己，前提是必须要有终止条件，否则会造成堆栈溢出。

js
function fn(n){
    if(n==1){
        return 1
    }
    // 返回1-5的阶乘  
    return n*fn(n-1);
};
fn(5);
console.log(fn(5));


/* n=5 返回 5*fn(4)  
   n=4 返回 5*(4*fn(3))
   n=3 返回 5*(4*(3*fn(2)))
   n=2 返回 5*(4*(3*(2*fn(1))))
   n=1 返回 5*(4*(3*(2*(1)))) 
*/
js
/* 斐波那契数列 0 1 1 2 3 5 8 13*/
// 1、for循环
function cir(n) {
    var n1 = 0,n2 = 1,num = 1;
    if (n == 1) {
        return 0;
    } else if (n == 2) {
        return 1;
    } else {
        for (var i = 3; i < n; i++) {
            num = n1 + n2;
            n1=n2; 
            n2=num;

        }
        return num;
    }
} cir(3);
console.log(cir(3));

// 2、递归函数(常用)
function recur(num) {
    if (num == 1) {
        return 0;
    } else if (num == 2) {
        return 1;
    }
    return recur(num - 1) + recur(num - 2);
} recur(7);
console.log(recur(7));

闭包函数 ​

把一个函数当作另一个函数的返回值，那么这个函数就是闭包函数

闭包函数优点：

  • 变量常驻内存
  • 避免全局变量被污染
  • 私有成员的存在

闭包函数的缺点：

由于常驻内存会造成内存资源浪费，使用不当会导致内存泄漏。

垃圾回收机制：

在函数中使用过的变量，在使用完成之后会被释放，但是在闭包函数中，变量常驻内存，所以闭包函数中的私有变量不会被垃圾回收机制回收。

js
function fn(){
    var a = 1;
    return function(){
        a++;
        return a
    }
}
fn()();  // 2
var newfn = fn(); // fn函数的返回值
newfn(); // 2
newfn(); // 3


function fn(n){
    return function fu(n2){
        return function fun(n3){
            return n*n2*n3
        } 
    }
}
var a=fn(3)(2)(4);
console.log(a);

构造函数 ​

通过 new 函数名 来实例化对象的函数叫构造函数。任何的函数都可以作为构造函数存在。之所以有构造函数与普通函数之分，主要从功能上进行区别的，构造函数的主要 功能为 初始化对象，特点是和new 一起使用。new就是在创建对象，从无到有，构造函数就是在为初始化的对象添加属性和方法。构造函数定义时首字母大写（规范）。

什么是new ​

new 申请内存, 创建对象,当调用new时，后台会隐式执行new Object()创建对象。所以，通过new创建的字符串、数字是引用类型，而是非值类型。

顶层：全局环境

Number（）将任意数据类型转换成整数 ​

  • 如果是布尔值，true为1，false为0
  • 如果是数值，转换为本身,会将无意义的后导零与前导零去掉
  • 如果为null，转换为0
  • 如果是undefined,转换为NaN（ not a number）
  • 如果是字符串，
    • 如果字符串中只有数字，则转换为数字（10进制）会忽略前导0和后导0
    • 如果是规范的浮点数，则转换为浮点数 会忽略前导0和后导0
    • 如果是空字符串，转换为0
    • 如果是其它值，转换为NaN

parselnt（）将任意数据类型转换成整数 ​

  • 如果一个字符串中只包含数字，转换为十进制数
  • 如果有多个空格，会先找到第一个非空的值进行转换，直到非数值时结束
  • 如果第一个值不是以数字、-数字、空格开头的，转换为NaN
  • 有两个参数时，第一个参数表示要转换的值，第二个参数表示几进制，返回值是一个十进制的数字
    • **注意：**第一个参数从最高位开始计算，只要有一位数可以识别为第二个参数传入的进制，则可以实现转化
    • **注意：**第二个参数可以传入的值为 2-36

parseFloat()将任意数据类型转换成浮点型 ​

  • 只有一个小数点起作用，其它无效
  • 如果字符串是一个有效的整数，他返回的是整数，不会返回浮点数。

String（）将任意数据类型转换成字符串 ​

  • 如果是null,undefined,转换为字符串 "null" "undefined"
  • 如果是数值类型，转换为本身的字符串，123 转换为 "123"
  • 如果是布尔类型，true为"true" false为"false"

Boolean（）将任意数据类型转换成布尔类型 ​

  • 转换结果是假：""(空串),null,undefined,0,false,NaN
  • 其它都为真

isNaN（）判断其参数是否是非数字值。 ​

  • 如果能转换成数值返回false，不能返回为true
  • 如果isNaN(x)返回false，那么x在任何算数表达式中都不会使表达式等于NaN；如果返回true，x会使所有算数表达式返回NaN

eval（"js代码"）运行字符串js代码 ​

  • 避免在不必要的情况下使用 eval

  • 需要解析，速度慢

  • 安全性差：所执行的字符串容易被操控修改从而收到攻击
