运算符 #

运算符包含7种：算术运算符、关系运算符、赋值运算符、逻辑运算符、三元运算符、一元运算符、特殊运算符。

算术运算符 #

js
+ 加或拼接, - 减, * 乘、/ 除, % 取余, 自增++ ， 自减-- ， 求幂**

  • ，可以进行运算，也可以进行拼接

    js
    var a="hello",b=710;
a+b="hello710"

  • 算术过程中如果可以转换,则转换成数字，转换不了转换成NaN

  • js
    var a=null,b=true,c=false
a+b==1;
a+c==0;
b+c==1;
b-a==-1

注意：任意数据与字符串相加都属于字符串拼接

  • 自增和自减
js
var num = 1;
console.log(num++);  //先输出后自增 1
console.log(++num);  //先自增后输出 3


console.log(num--);  //先输出后自减 1
console.log(--num);  //先自减后输出 -1

  • 求幂

    x**y 取x的y次方，同Math.pow(x, y)

    js
    3**2==9;

关系运算符 #

js
> 、 < 、 >= 、 ==等于 、 <= 、 !=不相等 、 ===全等 、 !==不全等

需要遵循以下规律：

返回的是boolean类型true或者false

  • 如果可以转换成数字，转换成数字进行比较，转换不了为false
  • true可以转换成1
  • false可以转换成0
  • null可以转换成0()

  1. 将比较的对象尝试转化为数字,能转化则按照数字比较,不能转化则为false

  2. NaN与任意数据比较结果均为 false

  3. 字符串和字符串比较, 比较第一个字符的unicode编码值，第一个字符要是相同，就比较第二个，依次往下(ASCII码表在最末尾)

    js
     '10000' < '2'   //1的unicode值比2的unicode值小  true       
 '10000' > 2    //转成数字比较  true

  4. ==和全等===的区别

    是只比较两边的值(如果两边类型不相同，则先尝试着转化，然后再比较)

    js
    0==undefined (false)
0==null (false)
NaN == NaN (false)
undefined==null (true)

    ===比较两边的值和类型，都相等才返回true；

  5. === 和 object.js()的区别

    js
    Object.is(+0,-0);               //false
+0===-0                         //true
Object.is(NaN,NaN);             //true
NaN===NaN                       //false

注意：字符串与字符串比较，比较的是ASCII表

js
== 和 ===的区别
== 只是值相等
===值相等，数据类型相等

undefined == null  //true
undefined ===null  //false
"10000"<"2"        //true
NaN == NaN         //false
null == false      //false
undefined == false //flase
NaN == false       //false
NaN == 0           //false
"" == false        //true
0 == false         //true

逻辑运算符 #

js
&& （与） 、 || （或）、 !（非）

！返回的是boolean类型true或者false

js
&& 
----与的表达式值是任意数据类型，返回的值取决于&&左右侧的真假
----&&左侧是真，结果取右侧； 左侧假，结果取左侧值；左侧真，右侧假，结果取右侧值
----如果两边条件都成立返回true，只要有一个为假返回false（前提是条件返回的值为布尔类型）

|| 
----||左侧是真，结果取左侧； 左侧假，结果取右侧值 ;两边都为假,结果取右侧值
----两边条件有一个为真返回true，两边都为假返回false（前提是条件返回的值为布尔类型）
!取反

undefined == null && null >=0  //true
undefined == null && null >0   //false
undefined == null || null >0   //true


!""             //true
!0              //true
!false          //true
!null           //true
!undefined      //true
!NaN            //true


短路原则：
- && 左边为假，右边条件不执行
- || 左边为真，右边条件不执行

一元运算符 #

js
typeof , instanceof , delete , new

只能操作一个值的操作符就叫做一元操作符

  • new (创建对象) delete (删除对象上的内容)
  • typeof -(负号) +(正号)
  • instanceof (判断对象是否由一个构造函数实例化)
