空间复杂度 ​

空间复杂度 (SpaceComplexity)是对一个算法在运行过程中临时占用存储空间大小的量度，记做S(n)。S(n)类似于时间复杂度,一个算法的空间复杂度S(n)定义为该算法所耗费的存储空间，它也是问题规模n的函数。

TIP

这里为了方便记忆，我将时间和空间复杂度统一为换成O(n),由于前端开发是属于重时间，轻空间的，所以这里就不过多阐述了。

O(1)有限的、可数的空间

js
//该函数的数量级为O(1) 存储的变量内存是可数的
function fn(arr=[]){
  const a=arr[0]
  const b=arr[1]
}

O(n)和输入的数据量有关的空间

js
//该函数的数量级为O(n) 存储的变量内存是根据arr长的决定的，有多长就存多少
function fn(arr=[]){
  cosnt arr2=[]
  for(let i=0;i<arr.length;i++){
    arr2[i]=arr[i]
  }
}
