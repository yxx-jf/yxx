CSS选择器 ​

添加样式时制定的规则，通过规则，能够选中页面中指定的元素。

  • 标签选择器：标签名是什么，选择器名称就是什么
html
标签名是div
那么选择器就是div{}

  • 类名选择器：在标签上添加class属性，设置类名（类名不能是数字开头、中文，必须是英文字母开头）

html
<div class="one"></div>
<style>
    .one{}
</style>

  • id选择器：在标签上添加id属性，设置id名（id名称不能是数字开头、中文、必须是英文字母开头）

html
<div id="name"></div>
<style>
    #name{}
</style>
  • 后代选择器：父选择器 子选择器{}
html
<div class="father">
    <div class="son"></div>
</div>
<style>
    .father{}
    .father .son{}
    .father div{}
</style>

  • 通配符选择器*{}

    通常写在样式里的最顶端

css
*{
margin:0;padding:0;
}

选择器 ​

选择器类别描述
*{}元素选择器通配符选择器，选中页面中所有的元素
div{} 元素选择器标签选择器，选中页面中所有指定标签名字的元素
.类名{}元素选择器类名选择器，例如.box{}，选择页面中指定类名的元素集合
#idname{} 元素选择器id选择器，例如#idname{}，选择页面中指定id名的元素，id是唯一的
div.box{}<br>box.aa{}<br>.box#idname{}元素选择器交叉选择器，描述的是同一个元素，选择页面中同时符合俩个条件元素集合
.box,#idname元素选择器群组选择器，多个选择器用‘，’隔开，选择多个选择器指定元素
父元素 子元素{}关系选择器后代选择器，例如.father .son{}， 选择所有被E元素包含的F元素。
.father>.son{} 关系选择器子包含选择器，只是父元素的第一层级,选择匹配F的元素，且该元素为所匹配E元素的子元素。
E +F 关系选择器相邻兄弟选择器，选择匹配F的元素，且该元素为所匹配E元素后面相邻的位置。
E~F关系选择器兄弟选择器,选择E元素后面的所有兄弟元素F，元素E与F必须同属一个父级。
E{attr}属性选择器选择匹配E的元素，且该元素定义了attr属性。E选择符可以省略，表示选择定义了attr属性的任意类型的元素。
E[attr="bar"] 属性选择器选择匹配E的元素，且该元素attr属性值为"bar"
E[attr~="bar"] 属性选择器选择匹配E的元素，且该元素定义了attr属性，attr属性值是一个以空格符分隔的列表，其中一个列表的值为"bar"，E选择符可以省略
E[attr!="en"]属性选择器选择匹配E的元素，且该元素定义了attr属性，attr属性值是一个用连字符（-）分隔的列表，值以"en"开头。
E[attr^="bar"]属性选择器选择匹配E的元素，且该元素定义了attr属性，attr属性值以"bar"开始。E选择符可以省略，表示可匹配任意类型的元素
E[attr$="bar"] 属性选择器选择匹配E的元素，且该元素定义了attr属性，attr属性值以"bar"结束。E选择符可以省略，表示可匹配任意类型的元素
E[attr*="bar"] 属性选择器选择匹配E的元素，且该元素定义了attr属性，attr属性值包含"bar"。E选择符可以省略，表示可匹配任意类型的元素
a:link{}伪类名选择器超链接未被访问的时候
div:active{}或a:active{} 伪类名选择器元素选中的时候
input:focus{} 伪类名选择器表单元素获取焦点
div:hover{} 伪类名选择器元素鼠标移入
a:visited{} 伪类名选择器超链接已被访问
E:first-child 结构伪类选择器选择匹配E的元素，且该元素为父元素的第一个子元素
E:last-child 结构伪类选择器选择位于其父元素中最后一个位置，且匹配E的子元素
E:nth-child(n) 结构伪类选择器选择所有在其父元素中第n个位置的匹配E的子元素
E:nth-last-child(n) 结构伪类选择器选择位于其父元素中且匹配E的最后第n子元素
E:nth-last-of-type(n)结构伪类选择器选择父元素中倒数第n个位置，且匹配E的子元素
E:first-of-type结构伪类选择器选择位于其父元素中且匹配E的第一个同类型的子元素。该选择器的功能类似于 E:nth-of-type(1)
E:last-of-type结构伪类选择器选择位于其父元素中且匹配E的最后第一个同类型的子元素。该选择器的功能类似于 E:nth-last-of-type(1)
E:only-child结构伪类选择器选择其父元素只包含一个子元素，且该子元素匹配E
E:only-of-type 结构伪类选择器选择其父元素只包含一个同类型的子元素，且该子元素匹配E
E:empty 结构伪类选择器选择匹配E的元素，且该元素不包含子节点
div::first-letter{}伪元素选择器第一个文本样式，选择段落中的第一个字母或中文字符
div::first-line{}伪元素选择器第一行文本样式， 选择文本中的首行
div::before{content:"",display:"元素属性"}伪元素选择器第一个文本样式之前，设置在对象前（依据对象树的逻辑结构）发生的内容。用来和content属性一起使用
div::after{content:"",display:"元素属性"}伪元素选择器第一个文本样式之后，设置在对象后（依据对象树的逻辑结构）发生的内容。用来和content属性一起使用

优先级是如何计算的⭐ ​

优先级就是分配给指定的 CSS 声明的一个权重，它由 匹配的选择器中的 每一种选择器类型的 数值 决定。

而当优先级与多个 CSS 声明中任意一个声明的优先级相等的时候，CSS 中最后的那个声明将会被应用到元素上。

当同一个元素有多个声明的时候，优先级才会有意义。因为每一个直接作用于元素的 CSS 规则总是会接管/覆盖（take over）该元素从祖先元素继承而来的规则。

选择器的优先级计算如下：

  • 计算选择器中ID选择器的数量（= a）
  • 计算选择器中的类选择器，属性选择器和伪类的数量（= b）
  • 计算选择器中标签选择器和伪元素的数量（= c）
  • 忽略通用选择器
选择器类型选择器
aID选择器
b类选择器、属性选择器、伪类选择器
c标签选择器、伪元素选择器
-通用选择器

将三个数字abc（在基数较大的数字系统中）连接起来可得出优先级。

css
*               /* a=0 b=0 c=0 -> specificity =   0 */
LI              /* a=0 b=0 c=1 -> specificity =   1 */
UL LI           /* a=0 b=0 c=2 -> specificity =   2 */
UL OL+LI        /* a=0 b=0 c=3 -> specificity =   3 */
H1 + *[REL=up]  /* a=0 b=1 c=1 -> specificity =  11 */
UL OL LI.red    /* a=0 b=1 c=3 -> specificity =  13 */
LI.red.level    /* a=0 b=2 c=1 -> specificity =  21 */
#x34y           /* a=1 b=0 c=0 -> specificity = 100 */
#s12:not(FOO)   /* a=1 b=0 c=1 -> specificity = 101 */
